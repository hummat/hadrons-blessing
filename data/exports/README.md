# Hadron's Blessing Exports

Checked-in JSON artifacts consumed by downstream projects (primarily BetterBots).

## Consumption model

**Primary:** Read files directly via filesystem (`../hadrons-blessing/data/exports/`).
**Secondary:** Regenerate via CLI commands in this repo.

## Artifacts

| File | Generated by | Consumer |
|------|-------------|----------|
| `bot-weapon-recommendations.json` | `npm run export:bot-weapons` | BetterBots #45 default profiles |

## Staleness

Each artifact has a `generated_at` timestamp. Regenerate after:
- Build re-extraction (new GL scrapes)
- Entity data changes affecting the export
- Before any BetterBots release consuming the data

## Adding new exports

Only add exports justified by a concrete downstream use case. See spec:
`docs/superpowers/specs/2026-03-15-betterBots-integration-contract-design.md`
